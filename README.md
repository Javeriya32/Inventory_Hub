
# InventoryHub

InventoryHub is a full-stack inventory management application built using Blazor WebAssembly and ASP.NET Core Minimal APIs.

## Features
- Front-end and back-end integration
- Debugged API routing and CORS issues
- Structured JSON responses with nested objects
- Optimized API communication

## Technologies
- Blazor WebAssembly
- ASP.NET Core Minimal API
- Microsoft Copilot
